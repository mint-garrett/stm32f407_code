
All my code from ELEC2850 where I use written structs copied from the STM32F407-Disc1 data map sections of the reference manual. This makes using pointers much easier, as the structs simplify the pointers.

I am mostly just blinking the on board LEDS (pd12,13,14,15) and an external LED using the button on the baord (PA0) and an external push button.
